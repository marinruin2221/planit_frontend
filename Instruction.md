
# Project Tech Stack

## Core Framework & Build Tool
- **React (v19.2.0)**: 사용자 인터페이스 구축을 위한 라이브러리
- **Vite (v7.2.6)**: 빠르고 가벼운 프론트엔드 빌드 도구
- **JavaScript (ES Modules)**: 주요 개발 언어

## Styling & UI
- **Tailwind CSS (v4.1.17)**: 유틸리티 퍼스트 CSS 프레임워크 (v4 최신 버전 사용)
- **Chakra UI (v3.30.0)**: 모듈화된 React 컴포넌트 라이브러리
- **Swiper (v12.0.3)**: 터치 슬라이더 및 캐러셀 구현
- **React Icons (v5.5.0)**: 다양한 아이콘 팩 제공

## Routing
- **React Router DOM (v7.10.0)**: SPA(Single Page Application) 라우팅 처리

## Utilities
- **Next Themes (v0.4.6)**: 테마(다크/라이트 모드) 관리
